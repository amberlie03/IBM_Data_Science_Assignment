## Project Overview
This project is a part of the IBM Data Science Professional Certificate, specifically the **Data Visualisation with Python module.** 

The goal of this project was to extract, clean, and visualise data. The final assignment investigated stock and revenue data for companies such as Tesla and GamStop using Python.
It combines data collection from APIs and web scraping with interactive visualisation to explore the relationships between stock prices and company revenue over time. 

## What This Project Demonstrates
- Using the yfinance **API**, I extracted historical stock data.
- I applied web scraping of revenue data from HTML pages using **BeautifulSoup.**
- Data was cleaned and formatted financial datasets using pandas.
- Combining multiple datasets for **comparative analysis.**
- Visualising stock price against revenue using plotly.
